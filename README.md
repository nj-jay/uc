# go-add

## function

仅用一个函数add实现的功能

理想的结果如下

```
//支持int int32 int64 string float32 float64类型
add(1, 2) = 3
add(1.1, 2.2) = 3.3
add("hello", "world") = "hello world"
add([]int{1, 2}, []int{4, 5, 6}) = []int{1, 2, 4, 5, 6}
add([]float32{1.1, 2.2}, float[32]{3.3, 4.4}) = float32{1.1, 2.2, 3.3, 4.3}

//后续需要用到哪些再进行补充
```
